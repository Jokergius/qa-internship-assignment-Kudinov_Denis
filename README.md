# Тестовое задание

## Описание проекта

В проекте используется автоматизированное тестирование с помощью инструмента **Playwright** для проверки навигации по сайту.  
Тестирование осуществляется в браузерах:
- Chromium
- Firefox
- Webkit

## Запуск тестов

1. Перейдите в директорию с тестами:
   ```bash
   cd ./automation
2. Запустите тесты:
    ```bash
    npx playwright test
